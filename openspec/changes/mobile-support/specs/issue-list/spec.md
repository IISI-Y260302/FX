## MODIFIED Requirements

### Requirement: 問題單列表顯示
系統 SHALL 依螢幕寬度以不同版面呈現問題單列表：桌面（≥ 768px）以表格顯示，手機（< 768px）以卡片列表顯示。已刪除的問題單預設不顯示。

#### Scenario: 桌面載入問題單列表（表格）
- **WHEN** 登入使用者以桌面瀏覽器進入列表頁
- **THEN** 系統以表格呈現問題單（編號、提出日期、提出人員、功能項目、嚴重度、狀態、類型）

#### Scenario: 手機載入問題單列表（卡片）
- **WHEN** 登入使用者以手機進入列表頁
- **THEN** 系統以卡片列表呈現問題單，每張卡片顯示編號、功能項目、嚴重度、狀態，點擊卡片導向詳細頁

#### Scenario: 管理員查看已刪除問題單
- **WHEN** 管理員勾選「顯示已刪除」篩選選項
- **THEN** 列表（表格或卡片）包含已刪除狀態的問題單，並以視覺方式（灰底）區分
